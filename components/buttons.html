--- layout: page title: Buttons ---

Probably the first thing people look at when choosing a CSS framework. The common button!

Basic Button

To make a button a Blaze button add the .c-button class to buttons, inputs or links.

Link 
<button class="c-button">Button</button>
<input type="button" class="c-button" value="Input">
<a class="c-button">Link</a>
<button class="c-button" disabled>Button</button>

Button Colours

There are four button colour modifiers .c-button--primary, .c-button--secondary, .c-button--success and .c-button--error. 

<button class="c-button">Button</button>
<button class="c-button c-button--primary">Button</button>
<button class="c-button c-button--success">Button</button>
<button class="c-button c-button--error">Button</button>

Button Groups

To group related buttons together you can use the .c-button-group wrapping element. This will bunch the buttons together and remove the spacing between them. 

<span class="c-button-group">
  <button class="c-button c-button--primary">Button</button>
  <button class="c-button c-button--secondary">Button</button>
</span>

Adding the .c-button-group--rounded modifier will make the first and last buttons rounded. 

<span class="c-button-group c-button-group--rounded">
  <button class="c-button c-button--primary">Button</button>
  <button class="c-button c-button--secondary">Button</button>
</span>
